# Start Here

This package contains two separate applications:

1. `backend/` — Laravel 13 REST API and administration backend.
2. `frontend/` — React 19 + Vite corporate website and React admin interface.

The supplied company logo is already included. The website uses the logo colours:

- Primary red: `#EF1717`
- Deep red: `#B80F16`
- Black: `#111111`
- White: `#FFFFFF`
- Soft background: `#F7F7F8`

Begin with `docs/06-INSTALLATION.md`, then read `docs/08-ADMIN-USER-GUIDE.md`.

Dependency folders such as `vendor` and `node_modules` are excluded. Install them with Composer and npm.
