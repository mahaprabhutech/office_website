# 11. Brand and Interface Guide

## Supplied logo

The original uploaded company logo is preserved in:

- `frontend/public/images/company-logo.png`
- `backend/public/images/company-logo.png`

Prepared web assets:

- `company-logo-transparent.png` — full horizontal logo with transparent background
- `brand-symbol.png` — circular eye symbol for hero, favicon and compact admin branding
- `favicon.png` — browser icon

## Colour system

| Token | Value | Use |
|---|---:|---|
| Primary red | `#EF1717` | Buttons, active navigation, highlights and status accents |
| Deep red | `#BD0808` | Hover and high-contrast red text |
| Black | `#111111` | Headings, dark sections and admin sidebar |
| White | `#FFFFFF` | Primary page surface |
| Soft grey | `#F7F7F8` | Alternating sections and admin background |
| Border grey | `#E7E7E7` | Cards, tables and form boundaries |

All reusable tokens are defined in `frontend/src/styles/theme.css`.

## Design principles

1. Strong black typography with restrained red highlights.
2. Circular visual elements inspired by the supplied brand symbol.
3. Generous spacing, rounded cards and clear content hierarchy.
4. Responsive navigation and forms for desktop, tablet and mobile.
5. Public website and admin panel share the same brand system while keeping different operational layouts.

## Logo safety

- Keep clear white space around the logo.
- Do not stretch, rotate or recolour the logo.
- Use the transparent horizontal logo on light backgrounds.
- Use the compact symbol only where horizontal space is limited.
- Obtain final brand approval before printing or trademark use.
