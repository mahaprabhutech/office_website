# 15. Implemented Scope

## Public React website

- Branded responsive header and footer
- Home page with hero, statistics, services, projects, workflow, articles and quotation call-to-action
- About, Services, Service Detail, Projects, Project Detail and Team pages
- Careers page with multipart resume application form
- Blog listing and article detail page
- Contact enquiry and detailed quotation forms
- Privacy Policy and Terms and Conditions starter pages
- Mobile navigation and responsive layouts
- API fallback content for design preview

## React administration panel

- Secure token login and logout
- Dashboard metrics and recent activity
- CRUD interface for services, projects, team members, jobs, articles and testimonials
- Enquiry search, follow-up, notes and status management
- Quotation review, estimate, proposal reference, notes and status management
- Job application review, resume download, notes and recruitment status
- Website settings
- Administrator account and role management
- Role-aware navigation

## Laravel REST API

- Laravel Sanctum token authentication
- Public content endpoints
- Contact, quotation and job application submission endpoints
- File validation and storage for attachments and resumes
- Admin CRUD controllers
- Dashboard analytics
- Module-level role middleware
- SQLite-ready migrations and seed data
- CORS configuration
- Models, relationships and casts
- API feature test starter

## Not bundled

- Composer `vendor/` dependencies
- npm `node_modules/` dependencies
- Production secrets and domain credentials
- Paid email, SMS, WhatsApp, map or analytics accounts
- Final lawyer-approved policy wording
- Production server configuration

These items must be installed or configured for the target environment using the included documents.
